README

ESRI shapefile layer depicting the Toronto District School Board's secondary schools catchement areas for 2015. Attibute data includes related school names, addresses, as well as the schools' EQAO results, literacy test results and Fraser Institute's ranking for 2015. This layer's projected coordinate system is NAD_1927_UTM_Zone_17N.

This shapefile was created as part of a pilot project looking to develop a "find my school app" for Toronto residents. 

IMPORTANT: Layer's polygon layer is not 100% complete.
