README

ESRI shapefile layer depicting the Toronto District School Boards secondary school's catchement areas for 2015. Attibute data includes related school names, addresses, as well as the schools' EQAO results, literacy test resulst and Fraser Institute's ranking for 2015. This layer's projected coordinate system is NAD_1927_UTM_Zone_17N.

This shapefile was created as part of a pilot project looking to develop a "find my school app" for Toronto residents. 

IMPORTANT: Polygon layer is not complete.
